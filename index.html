The UI Library we deserved, but never got.


TUI⚡ is a simple, modular and accessible component library that gives you the building blocks you need to build your applications.

Installation

Copy and paste the code in the head tag of your html document. Then you can use the components just by adding the classes to your html elements

< link rel = "stylesheet" href = "https://thunder-ui.netlify.app/styles.css" >
